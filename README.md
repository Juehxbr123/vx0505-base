# X (Twitter) в Telegram Бот-Ретранслятор

Простая, асинхронная и отказоустойчивая система ретрансляции постов, ответов и ретвитов из социальной сети X (Twitter) в указанный Telegram канал или чат.

## Преимущества решения
1. **Без платного API X.com**: Сбор данных идет через децентрализованную сеть стабильных шлюзов Nitter.
2. **Абсолютная автономность**: Работает на локальных JSON файлах, никакой развертки СУБД вроде PostgreSQL/Docker.
3. **Строгая защита от повторов**: Посты кэшируются атомарно, дублирование исключено.
4. **Запуск одной командой**: Логика бота и фонового воркера объединены в единое асинхронное событие.

## Установка и Старт

1. Склонируйте или распакуйте архив проекта.
2. Установите необходимые пакеты зависимостей (Если возникает ошибка 'Fatal error in launcher', используйте второй вариант):
```bash
# Вариант 1
pip install -r requirements.txt

# Вариант 2 (Для Windows с ошибками путей окружения)
python -m pip install -r requirements.txt
```

3. Откройте файл `.env` и настройте параметры (Токен бота, ID канала, Прокси).
4. Запустите скрипт:
```bash
python bot.py
```

## Команды управления в Telegram
- `/list` — список текущих целей
- `/add @username` — поставить на мониторинг (до 50 штук)
- `/delete username` — убрать из мониторинга
